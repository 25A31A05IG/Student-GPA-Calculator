# Student GPA Calculator (C)

## Description
A C program that calculates a student's GPA using marks and credit hours.

## Features
- Converts marks to grade points
- Calculates weighted GPA
- Displays final GPA

## Formula
GPA = (Σ(grade_point × credit_hours)) / Σ(credit_hours)

## Time Complexity
O(n)

## Space Complexity
O(n)
